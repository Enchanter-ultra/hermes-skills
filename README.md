# Hermes Skills - Enchanter-ultra

这是我的 Hermes Agent 技能合集。

## 使用方法

```bash
# 添加此仓库为技能源
hermes skills tap add Enchanter-ultra/hermes-skills

# 安装技能
hermes skills install openclaw-vision-fix
```

## 技能列表

| 技能 | 说明 |
|------|------|
| [openclaw-vision-fix](./openclaw-vision-fix/SKILL.md) | 诊断和修复 OpenClaw 图片识别（Vision）问题 |
